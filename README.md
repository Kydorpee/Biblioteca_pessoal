# Biblioteca Pessoal em Python

Este projeto foi desenvolvido apenas para praticar conceitos básicos de Python.

O sistema funciona no terminal e permite realizar operações simples de gerenciamento de uma biblioteca pessoal, como cadastrar, listar, buscar e remover livros.

Objetivos do projeto:

* Praticar lógica de programação
* Utilizar funções
* Trabalhar com listas e dicionários
* Manipular entradas do usuário
* Organizar código em Python

Tecnologia utilizada:

* Python 3

Execução do projeto:

```bash
python biblioteca.py
```

Projeto criado apenas para fins de estudo e prática de programação em Python.
